## Palindrom Kontrol Uygulaması

Bu proje, girilen kelime veya cümlenin **palindrom** olup olmadığını kontrol eden basit ve şık bir JavaScript uygulamasıdır.  

##  Canlı Demo


##  Özellikler

- ✔️ Girilen metnin palindrom olup olmadığını kontrol eder  
- ✔️ Boşluk, noktalama işaretleri ve harf büyüklüğünü yok sayar  
- ✔️ Hatalı veya boş girişte kullanıcıyı bilgilendirir  
- ✔️ Modern ve sade kullanıcı arayüzü  
- ✔️ Enter tuşu ile kontrol desteği  


##  Kullanılan Teknolojiler

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
